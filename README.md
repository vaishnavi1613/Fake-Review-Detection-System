# Fake Review Detection System

This project detects fake movie reviews using Machine Learning.

## Technologies Used
- Python
- FastAPI
- Streamlit
- SQLite
- Scikit-Learn
- SVM

## Features
- Home
- User Registration & Login
- Add Movies
- View All Movies
- Upload Movies
- Movie Management
- All Reviews
- Fake Review Detection
- Sentiment Reviews
- Ratings
- Trending Movies
- Like / Dislike System
- Logout
